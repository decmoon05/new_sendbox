## 🔧 기술 스택 최종 확정

### 프론트엔드
- **Framework**: Flutter 3.24+
- **Language**: Dart 3.3+
- **State Management**: Riverpod 2.5+
- **Local DB**: Isar 3.1+ (Hive 대안)
- **Network**: Dio 5.4+
- **Localization**: easy_localization 3.0+
- **Encryption**: crypto 3.0+, flutter_secure_storage

### 백엔드/클라우드
- **Primary**: Firebase
  - Firestore
  - Firebase Auth
  - Firebase Storage
  - Firebase Crashlytics
- **Secondary**: 자체 백엔드 (Node.js/Python)

### AI
- **Online**: Google Gemini API
- **Offline**: TensorFlow Lite

### 개발 도구
- **CI/CD**: GitHub Actions
- **Code Quality**: Dart Analyzer, lint
- **Testing**: flutter_test, mockito
